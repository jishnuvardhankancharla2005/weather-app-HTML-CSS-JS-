# Weather App
A clean and user-friendly weather forecast web application that allows users to search for current and hourly weather data for any city. The app provides instant weather updates, leverages location detection for convenience, and features a modern UI styled with Montserrat font.

# Features
City-based Search: Enter any city name for up-to-date weather details instantly.

Location Detection: Use the built-in location button to fetch weather data for your current location.

Current Weather: View current temperature, conditions, and weather icons for easy comprehension.

Hourly Forecast: Scrollable hourly weather updates for the next 24 hours, visually organized with weather icons.

Elegant UI: Sleek, mobile-friendly design with Montserrat typography, shadows, and responsive elements.

Error Handling: Clear feedback and helpful error messages if weather data cannot be retrieved.

# Technologies Used
Frontend: HTML5, CSS3 (with Google Fonts & Material Icons), JavaScript ES6

API: WeatherAPI for real-time weather data

Icons: SVG-based icons for weather conditions

# Getting Started
Prerequisites
Modern web browser (Chrome, Firefox, Edge, etc.)

Internet connection

# Usage
Type a city name and hit enter or click the search icon to view weather data.

Click "Detect my location" to allow browser-based geolocation for instant local weather.

Scroll horizontally for hourly weather details.
